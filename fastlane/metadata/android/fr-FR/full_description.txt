Découvrez Kai — votre alternative locale à OpenClaw dans votre poche. Une application de chat AI rapide et open source, propulsée par des modèles comme Llama, Gemini, GPT et DeepSeek. Plongez dans des conversations fluides pour du brainstorming, de l'écriture, des résumés, de l'apprentissage ou des discussions informelles — le tout dans une interface minimaliste qui se charge en quelques secondes.

Services AI pris en charge
• Ollama / LM Studio : Connectez-vous à vos modèles AI locaux auto-hébergés (compatible OpenAI)
• OpenAI : Accédez aux modèles GPT pour des conversations puissantes
• Gemini : L'AI avancée de Google avec prise en charge intégrée des pièces jointes
• DeepSeek : Modèles de raisonnement haute performance
• xAI : Modèles Grok innovants pour des réponses créatives et perspicaces
• Mistral AI : Modèles multilingues efficaces
• Groq : Inférence AI ultra-rapide pour des réponses instantanées
• OpenRouter : Accédez à des centaines de modèles via une seule API
• NVIDIA : Exécutez des modèles hébergés par NVIDIA
• Toute API compatible OpenAI : Utilisez votre propre point de terminaison

Fonctionnalités clés
• Mémoire persistante : L'AI retient les détails importants entre les conversations
• Stockage local chiffré : Conservez votre historique de chat en toute sécurité sur votre appareil
• Synthèse vocale : Écoutez les réponses de l'AI avec une voix naturelle
• Pièces jointes : Téléversez des fichiers pour enrichir vos conversations (pris en charge sur Gemini)
• Changement de service fluide : Basculez entre les fournisseurs AI sans effort en pleine conversation
• Conversations multi-tours : Discussions contextuelles où l'AI se souvient des messages précédents
• Daemon en arrière-plan et planification de tâches : Planifiez des tâches avec des expressions cron ou des intervalles de battement

Outils AI (Activer dans les paramètres)
• Heure locale : Date et heure actuelles précises
• Localisation : Réponses basées sur la localisation par détection IP
• Envoyer une notification : Notifications push déclenchées par l'AI
• Créer un événement : Ajoutez des événements à votre calendrier depuis le chat
• Recherche web : Laissez l'AI chercher sur le web des réponses à jour
• Exécuter une commande shell : Exécutez des commandes sur votre appareil via l'AI
• Planifier / Annuler / Lister les tâches : Gérez les tâches AI récurrentes

Confidentialité avant tout
Kai privilégie votre confidentialité : tout l'historique de chat est stocké localement avec un chiffrement robuste. Lors de l'utilisation de services AI cloud, vos données sont envoyées directement aux fournisseurs pour traitement — aucune collecte de données par l'application elle-même. Gardez le contrôle avec cette application de chat AI open source.
