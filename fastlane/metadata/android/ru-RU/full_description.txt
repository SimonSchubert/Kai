Познакомьтесь с Kai — вашей локальной альтернативой OpenClaw прямо в кармане. Быстрое приложение для AI-чата с открытым исходным кодом на базе моделей Llama, Gemini, GPT и DeepSeek. Погрузитесь в плавные диалоги для мозгового штурма, написания текстов, подведения итогов, обучения или простого общения — всё в минималистичном интерфейсе, который загружается за секунды.

Поддерживаемые AI-сервисы
• Ollama / LM Studio: Подключайтесь к собственным локальным AI-моделям (совместимым с OpenAI)
• OpenAI: Доступ к моделям GPT для мощных диалогов
• Gemini: Продвинутый AI от Google со встроенной поддержкой вложений файлов
• DeepSeek: Высокопроизводительные модели для сложных рассуждений
• xAI: Инновационные модели Grok для креативных и глубоких ответов
• Mistral AI: Эффективные мультиязычные модели
• Groq: Сверхбыстрая обработка AI для мгновенных ответов
• OpenRouter: Доступ к сотням моделей через единый API
• NVIDIA: Запуск моделей, размещённых на серверах NVIDIA
• Любой API, совместимый с OpenAI: Подключите собственную конечную точку

Основные функции
• Постоянная память: AI запоминает важные детали между разговорами
• Зашифрованное локальное хранилище: Безопасное хранение истории чатов на устройстве
• Озвучивание текста: Прослушивание ответов AI с естественным голосовым воспроизведением
• Вложение файлов: Загружайте файлы для улучшения бесед (поддерживается в Gemini)
• Плавное переключение сервисов: Переключайтесь между поставщиками AI прямо во время разговора
• Многоходовые диалоги: Контекстные чаты, в которых AI помнит предыдущие сообщения
• Фоновый демон и планирование задач: Планируйте задачи с помощью выражений cron или интервалов heartbeat

Инструменты AI (Включите в настройках)
• Узнать местное время: Точная информация о текущей дате и времени
• Узнать местоположение: Ответы с учётом местоположения через определение IP
• Отправить уведомление: Push-уведомления, инициированные AI
• Создать событие календаря: Добавляйте события в календарь прямо из чата
• Поиск в интернете: AI ищет в сети актуальную информацию
• Выполнить команду оболочки: Выполняйте команды на устройстве через AI
• Запланировать / Отменить / Показать задачи: Управляйте повторяющимися задачами AI

Конфиденциальность прежде всего
Kai ставит вашу конфиденциальность на первое место: вся история чатов хранится локально с надёжным шифрованием. При использовании облачных AI-сервисов данные отправляются напрямую поставщикам для обработки — приложение не собирает никаких данных. Сохраняйте контроль с этим AI-чатом с открытым исходным кодом.
