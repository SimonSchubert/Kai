Kai를 만나보세요 — 주머니 속 로컬 OpenClaw 대안 앱. Llama, Gemini, GPT, DeepSeek 등의 모델을 지원하는 빠르고 오픈소스인 AI 채팅 앱입니다. 브레인스토밍, 글쓰기, 요약, 학습, 일상 대화까지 — 몇 초 만에 로딩되는 미니멀한 인터페이스에서 매끄러운 채팅을 즐기세요.

지원 AI 서비스
• Ollama / LM Studio: 자체 호스팅 로컬 AI 모델에 연결 (OpenAI 호환)
• OpenAI: 강력한 대화를 위한 GPT 모델 이용
• Gemini: 파일 첨부를 지원하는 Google의 고급 AI
• DeepSeek: 고성능 추론 모델
• xAI: 창의적이고 통찰력 있는 응답을 제공하는 혁신적인 Grok 모델
• Mistral AI: 효율적인 다국어 모델
• Groq: 빠른 응답을 위한 초고속 AI 추론
• OpenRouter: 하나의 API로 수백 개 모델에 접근
• NVIDIA: NVIDIA에서 호스팅하는 모델 실행
• OpenAI 호환 API: 자신만의 엔드포인트 사용 가능

주요 기능
• 영구 메모리: AI가 대화 간 중요한 세부 정보를 기억
• 암호화된 로컬 저장소: 기기에 채팅 기록을 안전하게 저장
• 텍스트 음성 변환: 자연스러운 음성으로 AI 응답을 들을 수 있음
• 파일 첨부: 파일을 업로드하여 대화를 풍부하게 (Gemini에서 지원)
• 원활한 서비스 전환: 대화 중에도 AI 제공자를 손쉽게 변경
• 멀티턴 대화: AI가 이전 메시지를 기억하는 맥락 인식 채팅
• 백그라운드 데몬 및 작업 예약: cron 표현식 또는 하트비트 간격으로 작업 예약

AI 도구 (설정에서 활성화)
• 현지 시간 확인: 정확한 현재 날짜 및 시간 정보
• 위치 확인: IP 감지를 통한 위치 기반 응답
• 알림 전송: AI가 트리거하는 푸시 알림
• 캘린더 이벤트 생성: 채팅에서 캘린더에 일정 추가
• 웹 검색: AI가 웹에서 최신 답변을 검색
• Shell 명령 실행: AI를 통해 기기에서 명령어 실행
• 작업 예약 / 취소 / 목록: 반복 AI 작업 관리

프라이버시 최우선
Kai는 프라이버시를 최우선으로 합니다. 모든 채팅 기록은 강력한 암호화로 기기에 로컬 저장됩니다. 클라우드 AI 서비스 사용 시 데이터는 처리를 위해 제공자에게 직접 전송되며, 앱 자체에서는 어떠한 데이터도 수집하지 않습니다. 이 오픈소스 AI 채팅 앱으로 데이터를 직접 관리하세요.
